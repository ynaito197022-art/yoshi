# yoshi

GitHubでアプリ開発を始めるための初期リポジトリです。

## 現在の内容

- `index.html`  
  HTML / CSS / JavaScript を1ファイルにまとめたシンプルな初期ページです。

## 使い方

1. `index.html` を開く
2. 必要に応じてHTML、CSS、JavaScriptを編集する
3. アプリの内容が固まってきたら、ファイルを分割する
